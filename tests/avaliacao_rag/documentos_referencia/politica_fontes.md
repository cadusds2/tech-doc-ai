# Política de fontes

Toda resposta RAG deve citar as fontes recuperadas, incluindo nome do arquivo e identificador do trecho usado.

Se a pergunta não tiver contexto suficiente nos documentos indexados, o sistema deve informar que não encontrou contexto seguro em vez de inventar uma resposta.

As fontes úteis são aquelas que contêm evidências diretas para a pergunta do usuário.
